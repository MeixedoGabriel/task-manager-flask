# Task Manager Flask

Sistema web de gerenciamento de tarefas desenvolvido com Python, Flask e SQLite.

## Funcionalidades

- Criar tarefas
- Concluir tarefas
- Deletar tarefas
- Dashboard
- Filtros
- Pesquisa
- Banco de dados SQLite

## Tecnologias

- Python
- Flask
- SQLite
- HTML
- CSS
- Git
- Render
